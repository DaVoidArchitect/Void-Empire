# CONTRIBUTING TO PRIMAL ORIGINS

**Welcome Contributors!** 🎉

Primal Origins is an open-source SoC IP Core. Once released, all development happens through community contributions.

---

## 🌟 HOW TO CONTRIBUTE

### Quick Start
1. **Fork** the repository
2. **Clone** your fork
3. **Create** a branch (`git checkout -b feature/your-feature`)
4. **Make** your changes
5. **Test** thoroughly
6. **Commit** (`git commit -m 'Add feature'`)
7. **Push** (`git push origin feature/your-feature`)
8. **Open** a Pull Request

---

## 📋 CONTRIBUTION GUIDELINES

### Code Standards
- **Language:** SystemVerilog-2012
- **Style:** Follow existing code conventions
- **Comments:** Document complex logic
- **Synthesizable:** All code must synthesize

### Testing
- ✅ Add testbenches for new features
- ✅ Ensure all tests pass
- ✅ Maintain >90% code coverage
- ✅ Update documentation

### Documentation
- Update inline comments
- Update relevant docs in `docs/`
- Add to `CHANGELOG.md`
